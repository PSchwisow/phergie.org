--- title: 'New Unit Testing Docs' date: '2010-09-29 14:15:54' layout: post slug: new-unit-testing-docs categories: - Announcements tags: { } --- The Developers page now includes a section on unit testing Phergie plugins. Efforts to improve code coverage of the Phergie codebase continue, so if you'd like to help out, find a plugin lacking a unit test suite and write one. If you're developing your own plugins to contribute back to the project, try to include unit test suites for them when you submit them. Our continued thanks go out to all of our contributors.
